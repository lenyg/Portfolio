Cash Register

This project will generate an alert box, be sure your have enabled pop ups!

See the Pen cashRegister by Lynnette Griffen (@lenyg) on CodePen.

In this project we were instructed to use Javascript to write a function called cashRegister that takes a shopping cart object. The object will contain item names and prices (itemName: itemPrice). The function should return the total price of the shopping cart. I strengthened my profeciency of JavaScript by demonstrating my knowledge of objects, including how to access and modify them. Mastering the use of objects and functions will improve my ability to use JavaScript to create a truly dynamic experience.
